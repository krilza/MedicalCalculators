# MedicalCalculators
# Разработка приложения с набором медицинский калькуляторов
## Список медицинских калькуляторов:
### Калькулятор ИМТ (Индекс массы тела)
   - Вх. данные: *рост, вес*
   - Формулы:
      - ИМТ: Вес / ((Рост / 100)**2)
      - Норма веса для конкретного роста:
        - Мин: 18.5 * (Рост / 100)**2)
        - Макс: 24.9 * (Рост / 100)**2)
   - Вычисляет и выводит ИМТ
   - В соответствии с ИМТ выводит одну из четырёх категорий:
      - **Недостаточный вес:** ИМТ < 18.5
      -  **Нормальный вес:** ИМТ < 25
      -  **Избыточный вес:** ИМТ < 30
      -  **Ожирение:** ИМТ > 30
   - Также выводит норму веса для введённого роста
### Калькулятор ППТ (Площадь поверхности тела)
   - Вх. данные: *рост, вес*
   - Формула: Корень из ((рост * вес) / 3600) (Формула Мостеллера)
   - Вычисляет ППТ
### Калькулятор САД (Среднее артериальное давление)
   - Вх. данные: *Диастолическое, систалическое давления*
   - Вычисляет САД
   - Формула: Диастолическое + (Систалическое - диастолическое) / 3
   - Сравнивая результат вычисления со средним нормальным давление выводит одно из трёх заключений:
      - **Низкое:** 0 - 69
      - **Норма:** 70 - 100
      - **Высокое:** 100+
